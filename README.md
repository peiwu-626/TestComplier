# 語法：
* 冪次運算： base  ^  n
```diff
calc > 2^3 
8
```
* 根號運算： num  **  n
```diff
calc > 4**2 
2.0
```
* for-loop： for 變數 loop 次數 : ("欲執行的指令運算");
```diff
calc > num=0
calc > for num loop 3 : i=i+2;
calc > num
6
``` 
* if-else： if ("條件判斷") : "expression" ; else : "expression" ; 
```diff
calc > i=0
calc > if(i<0):i=1;else:i=2;
calc > i
2
```
